# GitFame

**GitFame** — это консольная утилита для подсчёта статистики авторов Git-репозитория. Утилита анализирует количество строк, коммитов и файлов, затронутых каждым автором, и выводит результат в удобном формате.

## Возможности

- Подсчёт количества строк, коммитов и файлов для каждого автора.
- Сортировка результатов по количеству строк, коммитов или файлов.
- Поддержка различных форматов вывода (табличный, CSV, JSON).
- Фильтрация по расширениям файлов и языкам программирования.
- Отображение прогресса во время вычислений.

## Установка

Для установки утилиты, выполните следующие команды:

```bash
go install ./gitfame/cmd/gitfame/
```
## Использование

Вы можете использовать утилиту с различными флагами для настройки её поведения. Например:

gitfame --repository=. --extensions='.go,.md' --order-by=lines

## Опции:

--repository=DIR — путь к директории с репозиторием (по умолчанию текущая директория .).

--revision=REVISION — указатель на коммит (по умолчанию HEAD).

--order-by=KEY — сортировка результатов по ключу: lines, commits, files (по умолчанию lines).

--use-committer — использовать коммиттера вместо автора.

--format=FORMAT — формат вывода: tabular (по умолчанию), csv, json, json-lines.

## Пример вывода
Name                   Lines Commits Files
Joe Tsai               12154 92      49
colinnewell            130   1       1
Roger Peppe            59    1       2
A. Ishikawa            36    1       1
Tobias Klauser         33    1       2
Форматы вывода:

tabular: человекочитаемый формат с выравниванием.

csv: данные в формате CSV.

json: данные в формате JSON.

json-lines: данные в формате JSON для каждого результата на новой строке.
